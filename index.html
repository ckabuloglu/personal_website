SnifferGPS iOS App

I designed and built an app for my startup SnifferGPS. The app has ability to track down a Sniffer device and show it on Google Maps. Besides that, the users can create and edit their profiles, add dogs and devices to their profile to track them later and change tracking settings. This was my first encounter with Swift and I enjoyed it so far. I am looking forward to get better and create more iOS apps.

SnifferGPS Webapp

I designed and built the web app version of the SnifferGPS's iOS app as well using mainly bootstrap and javascript for front-end interactions and as well as PHP and MySQL as back-end and data storage purposes. The features of the app are tracking devices on the Google Maps API, create and edit profiles, add dogs and change various tracking settings.
